
# 🚀 CareerPilot – AI Powered Interview & Career Strategy Platform

> Your personal AI career coach for interview preparation, resume building, and career growth.

---

## 📌 Overview

CareerPilot is an AI-powered platform designed to help students and professionals prepare for interviews, improve resumes, and build a strong career strategy.

It provides personalized interview questions, AI feedback, resume analysis, and career roadmaps — all in one place.

## ✨ Features

- 🤖 AI Interview Preparation (role-based questions + AI feedback)
- 📄 Resume Analyzer (ATS optimization + suggestions)
- 🎯 Career Roadmap Generator (goal-based learning path)
- 💬 AI Career Chatbot (instant career guidance)
- 📊 Performance Tracking (progress + analytics)

---

## 🛠️ Tech Stack

Frontend:
- React.js
- Tailwind CSS
- Axios

Backend:
- Node.js
- Express.js

Database:
- MongoDB

AI:
- OpenAI API

---

## 🏗️ Architecture

Client (React) → API (Express.js) → Backend Logic → MongoDB → AI Services

## ⚙️ Installation & Setup

### 1. Clone Repository

```bash
git clone https://github.com/Rahul7007/Car9eerPilot.git
cd CareerPilot

 2. Install Dependencies

cd frontend
npm install
npm run dev

Backend:

cd backend
npm install
npm run dev

3. Environment Variables

Create a .env file in backend folder:
PORT=3000
MONGO_URI=your_mongodb_connection
OPENAI_API_KEY=your_api_key
JWT_SECRET=your_secret
